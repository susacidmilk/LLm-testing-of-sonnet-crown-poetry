# 十四行花环诗组编写能力测试

本测试旨在评估不同大语言模型在 **十四行花环诗（Crown of Sonnets / Sonnet Redoublé）** 这一复杂诗歌形式上的创作能力。

## 测试规则

统一提示词要求：

- 花环诗组由 **14 首十四行诗** 与 **1 首"冠顶诗"** 组成
- 前 14 首中，每一首的末行即为下一首的首行
- 第 14 首的末行必须回到第 1 首的首行，形成闭环
- 第 15 首"冠顶诗"由前 14 首的首行依次组成
- 全组押**莎士比亚十四行诗韵式**：`abab cdcd efef gg`，各诗独立押韵
- 如无特殊说明，测试方式为各模型官方网页直接对话
